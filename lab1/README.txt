# User Validation System (Java)

This project was developed as part of a Software Engineering course.

## 📌 Description

The program reads a file of users, where each line contains a username (email) and password.
It validates each entry and stores only valid users.

## ✅ Features

* Email validation based on strict formatting rules
* Password validation (length, digits, letters, symbols)
* Object-Oriented design using a `User` class
* File reading using `Scanner`
* Stores valid users using `ArrayList`
* Sorts users alphabetically by username

## ▶️ How to Run

```bash
javac User.java UsersApp.java
java UsersApp
```

## 📂 Input Format

Each line in `users.txt`:

```
username password
```

## 📤 Output

* Only valid users are printed
* Output is sorted alphabetically

## 🛠 Technologies

* Java
* OOP principles
* File I/O
* Collections (ArrayList, Comparator)
